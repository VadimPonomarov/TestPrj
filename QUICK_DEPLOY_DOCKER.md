# QUICK DEPLOY DOCKER / ШВИДКИЙ ДОКЕР-ДЕПЛОЙ

## English
1. Ensure Docker Desktop/Engine (v24+) with Compose V2 is running.
2. From the project root run:
   ```powershell
   python deploy.py
   ```
3. Follow the prompts (if any). The script waits for PostgreSQL/Django and shows follow-up commands.

## Українська
1. Переконайтесь, що Docker Desktop/Engine (v24+) з Compose V2 запущений.
2. У корені проєкту виконайте:
   ```powershell
   python deploy.py
   ```
3. Дотримуйтесь підказок (за потреби). Скрипт чекає готовність PostgreSQL/Django і показує наступні кроки.
